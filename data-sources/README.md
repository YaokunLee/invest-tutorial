# Data Sources Archive

This folder stores source screenshots and raw reference materials used while preparing articles.

## Folder conventions

- `employee-counts/`: charts or screenshots related to company employee counts.
- `employee-counts/internet-platforms/`: employee-count charts for internet platform companies.
- `financials/`: API extracts or derived financial-statement datasets used to build article charts.

Keep article-ready public assets under `public/invest/`. Keep raw source screenshots and working references here.
