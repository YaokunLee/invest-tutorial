## 课程定位

这门课不是金融学院教材，也不是短线交易教程。

目标读者是金融基础较弱、但希望认真学习美股长期投资的人。课程要帮助读者建立四种能力：

1. 看懂财报：知道收入、利润、现金流、资本开支、股权激励这些指标到底说明什么。
2. 研究公司：能拆业务、拆收入、看竞争格局、判断增长质量。
3. 研究行业：能画产业链、看利润流向、判断周期、识别谁在产业链里有定价权。
4. 做投资判断：能把公司质量、行业阶段、宏观周期、估值和投资心态放在一起考虑。

写作原则：

- 多用真实公司案例，不只讲概念。
- 多放图，尤其是财报 PPT、公司财报截图、自己整理的数据图。
- 每篇文章最后尽量给练习题，让读者能自己动手。
- 不追求学术完美，但要让初学者建立正确的投资 sense。

## 课程总结构

### 第一部分：投资入门和学习框架

目标：先告诉读者为什么要学投资、这门课怎么学、长期投资和短线交易有什么区别。

已完成：

- `src/pages/index.mdx`：介绍、为什么要学投资、课程目录。
- `src/pages/investing-vs-trading.mdx`：为什么投资不是 Trading（交易）。明确这门课不评价、不教授 trading；如果读者想学 trading，建议去 YouTube 搜相关内容；本课只聚焦长期投资框架。

后续可补：

- 投资到底赚的是什么钱：企业价值增长、估值变化、分红回购、汇率和流动性。
- 投资前的基本心态：不追求一夜暴富、不用杠杆、不把观点当信仰。

### 第二部分：财报基础

目标：让读者能读懂公司最基本的财务报表，并知道不同指标在投资中怎么用。

已完成：

- `src/pages/income-statement.mdx`：如何读懂收入表。
- `src/pages/company-lifecycle.mdx`：从收入增速判断公司所处阶段。

后续建议文章：

1. 如何读懂现金流量表
   - 重点讲 operating cash flow、CapEx、free cash flow、SBC。
   - 案例：META、Alphabet、Microsoft、Amazon。
   - 必讲：为什么 AI 公司现在必须看 CapEx 和 FCF。

2. 如何读懂资产负债表
   - 现金、债务、应收账款、库存、商誉、租赁负债。
   - 案例：Apple 的现金和回购、半导体公司的库存周期、Microsoft 的 lease commitments。

3. 毛利率、经营利润率和净利率怎么用
   - 毛利率看定价权和商业模式。
   - 经营利润率看规模效应。
   - 净利率要结合税、投资收益、一次性项目。
   - 案例：NVIDIA、META、Google Cloud、SaaS 公司。

4. 股权激励 SBC 为什么重要
   - SBC 为什么会让 GAAP 利润和 FCF 看起来更好。
   - 高成长公司如何用 SBC “支付工资”。
   - 案例：SaaS、AI 初创上市公司、META 控费后的利润改善。

### 第三部分：如何研究一家公司

目标：建立公司研究框架，从一句话商业模式开始，拆业务、拆收入、看竞争、看管理层、看估值。

已完成：

- `src/pages/company-research.mdx`：如何研究一家公司。
- `src/pages/microsoft-case.mdx`：案例：如何研究 Microsoft。
- `src/pages/alphabet-case.mdx`：案例：如何研究 Alphabet（Google）。

当前已写公司的重点：

- Microsoft：Microsoft 365、Azure、More Personal Computing、AI CapEx、RPO、企业客户护城河。
- Alphabet：Search、YouTube、Google Cloud、订阅、Other Bets、AI 对搜索和 CapEx 的影响。

后续建议案例：

1. Amazon 案例
   - 核心不是电商收入，而是 AWS 利润、广告业务、Prime 生态、物流效率。
   - 重点讲：低利润零售 + 高利润 AWS + 广告变现。
   - 可对比 Microsoft Azure 和 Google Cloud。

2. NVIDIA 案例
   - 产业链位置：AI 基础设施卖铲人。
   - 重点讲：GPU、CUDA、网络、客户集中度、云厂商自研芯片风险。
   - 必讲：高毛利本身会激励客户寻找替代方案。

3. Meta 案例
   - 广告现金牛、Reality Labs、AI 推荐系统、CapEx、回购。
   - 重点讲：从 2022 年估值崩塌到效率年，再到 AI CapEx 周期。

4. Apple 案例
   - 硬件、服务、生态、回购、增长放缓后的估值问题。
   - 重点讲：好公司但增长有限时，估值怎么判断。

5. SaaS 公司案例
   - 可选：Snowflake、Datadog、HubSpot、Zeta。
   - 重点讲：收入留存率、销售效率、Rule of 40、SBC、EV/Sales。

### 第四部分：如何研究一个行业

目标：让读者从单家公司跳到产业链视角，知道一家公司赚钱不是孤立发生的。

已完成：

- `src/pages/industry-research.mdx`：如何研究一个行业。
- `src/pages/cloud-ai-capex-cycle.mdx`：案例：云厂商 AI CapEx 周期：AI 基建到底是不是泡沫？

后续建议文章：

1. AI 产业链怎么拆
   - 上游：芯片、HBM、半导体设备、先进封装、光通信、电力。
   - 中游：云厂商、模型公司、数据中心。
   - 下游：企业软件、AI 应用、消费端产品。
   - 重点讲：同样看好 AI，不代表只能买 NVIDIA。

2. 半导体周期怎么研究
   - 库存、产能利用率、价格、CapEx、订单周期。
   - 案例：存储、先进封装、半导体设备。

3. 软件行业怎么研究
   - SaaS、企业软件、AI Agent、销售效率、客户留存。
   - 案例：Microsoft、ServiceNow、Datadog、HubSpot。

4. 医疗/医疗器械行业怎么研究
   - 市场规模、渗透率、临床证据、报销、销售渠道。
   - 案例可后续从已有投资研究里选。

### 第五部分：宏观周期和美联储

目标：让读者理解利率、通胀、流动性对估值和资产价格的影响。

已提到但未系统写：

- 美联储加息/降息。
- 周期和估值的关系。

后续建议文章：

1. 美联储加息降息到底影响什么
   - 利率影响折现率、估值倍数、融资成本、风险偏好。
   - 高成长股为什么对利率更敏感。
   - 案例：2022 年科技股估值压缩，2023-2026 AI 反弹。

2. 通胀、就业和经济周期怎么看
   - CPI、PCE、失业率、工资、PMI。
   - 不追求宏观预测，而是理解市场为什么重新定价。

3. 为什么估值会周期性扩张和收缩
   - 同样一家公司，为什么有时候 20 倍 PE 合理，有时候 30 倍也合理。
   - 重点讲利率、增长预期、风险偏好。

4. 流动性和市场情绪
   - 不做短线，但要知道市场为什么会极度乐观或悲观。
   - 可以结合 fear & greed、VIX、信用利差。

### 第六部分：估值方法

目标：让读者知道不同公司适合不同估值方法，不要机械套 PE。

已完成：

- `src/pages/dcf.mdx`：如何给一家公司估值：DCF。

后续建议文章：

1. PE、PS、EV/Sales、EV/EBITDA 分别什么时候用
   - 成熟盈利公司：PE。
   - 未盈利高成长公司：PS / EV/Sales。
   - 资本结构差异较大：EV/EBITDA。
   - 现金流稳定公司：DCF。

2. DCF 的真正用途
   - DCF 不是为了算出精确股价，而是为了拆预期。
   - 增速、利润率、CapEx、折现率、终值哪个最敏感。

3. 如何用情景分析估值
   - Base / Bull / Bear 三种情景。
   - 重点不是猜准，而是知道赔率和风险。

4. 如何看市场预期
   - 股价隐含了什么增长？
   - 什么样的财报会让市场上调或下调预期？

### 第七部分：投资心态、组合和风险控制

目标：避免读者只会分析公司，但真正投资时被情绪和仓位打败。

后续建议文章：

1. 买入前要问自己的问题
   - 我买的是公司质量、行业趋势、估值修复，还是短期情绪？
   - 如果跌 30%，我会加仓、持有还是卖出？

2. 怎么处理亏损
   - 判断是 thesis broken，还是市场短期波动。
   - 什么时候认错，什么时候忍受波动。

3. 仓位管理
   - 核心仓、观察仓、事件仓。
   - 不把单一观点变成不可承受风险。

4. 好公司不等于好股票
   - 估值太贵时，好公司也可能长期不赚钱。
   - 案例：2021 年高估值 SaaS，2022 年科技股回撤。

5. 如何复盘自己的投资
   - 写买入理由、卖出理由、跟踪指标。
   - 区分运气和能力。
