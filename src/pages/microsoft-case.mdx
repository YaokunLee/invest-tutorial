# 案例：如何研究 Microsoft

> 数据口径：除非特别说明，本文使用 Microsoft FY2025 年报，以及 FY2026 Q3 财报。Microsoft 的财年截止日是 6 月 30 日，所以 FY2025 对应的是 2024 年 7 月到 2025 年 6 月。

Microsoft 是非常适合新手练习公司研究的案例，因为它不是单一产品公司。它同时有企业软件、云计算、AI 基础设施、开发者工具、游戏、搜索广告、Windows 等业务。

如果你只说“Microsoft 是 AI 公司”或者“Microsoft 是卖 Office 的公司”，都不完整。

更准确的一句话应该是：

> Microsoft 是一家以企业客户为核心的软件和云平台公司，通过 Microsoft 365、Azure、Windows、Dynamics、LinkedIn、GitHub、Xbox 等产品，把企业和个人的工作流、开发流、云基础设施和 AI 使用场景连接在一起。

这句话里有几个关键词：**企业客户、软件订阅、云平台、AI、工作流**。这些词决定了 Microsoft 的商业模式质量。

## 先看整体：Microsoft 到底靠什么赚钱

Microsoft 把业务分成三大板块。先不要急着看数字，先把这三个名字翻译成人话：

1. **Productivity and Business Processes**：主要是 Microsoft 365、Office、Teams、LinkedIn、Dynamics 365。你可以把它理解成“企业办公软件和企业应用”。
2. **Intelligent Cloud**：主要是 Azure、服务器产品、企业服务、GitHub cloud services 等。你可以把它理解成“云基础设施和企业 IT 平台”。
3. **More Personal Computing**：主要是 Windows、设备、Xbox / Gaming、Search and news advertising。你可以把它理解成“个人电脑入口、游戏和搜索广告”。

这一步很重要。因为如果你不知道每个板块里装的是什么，只看表格里的英文名字，很容易得出空泛结论。

尤其是 More Personal Computing。它不是一个无关紧要的小业务。早期 Microsoft 的核心增长来自 PC 普及，Windows 和 Office 跟着 PC 出货量一起扩张；所以 Windows 所在的个人计算入口，曾经是 Microsoft 最重要的增长来源之一。但今天再看 Microsoft，问题已经变成：这块业务还是不是主要增长引擎？

带着这个问题，再看 FY2025 年报里的三大板块：

| 业务板块 | FY2025 收入 | 同比增速 | FY2025 经营利润 | 经营利润率 |
| --- | ---: | ---: | ---: | ---: |
| Productivity and Business Processes | $120.8B | +13% | $69.8B | 57.8% |
| Intelligent Cloud | $106.3B | +21% | $44.6B | 42.0% |
| More Personal Computing | $54.6B | +7% | $14.2B | 25.9% |
| **合计** | **$281.7B** | **+15%** | **$128.5B** | **45.6%** |

有了业务含义，再读这个表，你应该先得到三个结论：

1. **Microsoft 不是只靠 Azure。** Productivity and Business Processes 仍然是收入和利润最大的板块，说明 Microsoft 的底盘还是企业软件和企业应用。
2. **Azure 所在的 Intelligent Cloud 是当前最重要的增长板块。** FY2025 收入增长 21%，明显高于公司整体 15% 的增速，后面 FY2026 Q3 还会看到它继续加速。
3. **More Personal Computing 已经不是核心增长引擎。** 这个结论不是因为它“不重要”，而是因为它的收入规模最小、增速最低、利润率也低于另外两个板块。它更像成熟现金流、PC 生态入口、Gaming 和 Search 的战略补充，而不是今天 Microsoft 增长故事的主线。

这就是公司研究里的第一步：不要被一个标签带跑，先看收入和利润到底来自哪里。

上面的 FY2025 年报数字告诉我们 Microsoft 的长期结构；接下来要看 FY2026 Q3，是为了确认这条结构在最新季度有没有变化。读 Microsoft 的主线不是“AI 很热”，而是三件事：Productivity and Business Processes 提供高利润现金流，Intelligent Cloud 承担主要增长，More Personal Computing 继续扮演生态入口和周期性补充。

FY2026 Q3 的数据延续了这条主线：Microsoft 仍然不是单点增长，而是用高利润的软件订阅支撑现金流，用 Azure 和 AI 拉动增长，同时用 Windows、搜索、游戏维持生态入口。

## 业务线一：Productivity and Business Processes

这个板块可以理解成 Microsoft 的“企业生产力软件帝国”。

它主要包括：

- Microsoft 365 Commercial：企业版 Office、Teams、Exchange、SharePoint、Microsoft 365 Copilot、安全与合规产品等
- Microsoft 365 Consumer：个人版 Office 订阅等
- LinkedIn：招聘、广告、会员、销售线索
- Dynamics 365：CRM、ERP、Power Apps、Power Automate 等企业应用

FY2025 这个板块收入 $120.8B，同比增长 13%；经营利润 $69.8B，经营利润率接近 58%。这说明它是一门极好的生意。

这张 segment overview 更直观：Productivity and Business Processes 不只是收入增长，经营利润也在增长，而且利润率非常高。研究这类业务时，要看收入增长有没有带来<FinanceTerm term="利润杠杆" />。

为什么好？

### 1. 它嵌在企业工作流里

Office、Outlook、Teams、SharePoint、Excel、PowerPoint 不是普通工具，而是企业每天都要用的工作流。

一家公司一旦深度使用 Microsoft 365，迁移成本非常高。员工习惯、文件格式、权限系统、会议系统、邮箱系统、协同流程，全都绑在里面。

这就是 Microsoft 的护城河：不是用户“喜欢”它，而是企业“很难离开”它。

> **思考一下：迁移成本也是护城河**
>
> 企业换掉 Microsoft 365，不只是换一个软件，还要迁移邮件、文档、权限、会议系统和协作流程，员工也要重新学习。
>
> 所以迁移成本高，本身就是一种护城河。类似的例子还有 Adobe、Salesforce、SAP：客户越依赖它，越难离开，公司也就更容易保持高续费率和较高利润率。

### 2. 它有持续提价能力

Microsoft 365 的商业模式从一次性软件授权，逐渐变成订阅。订阅的好处是收入更稳定，而且可以通过高级版本、安全功能、Copilot、Teams Phone、Power Platform 等模块持续提高 <ArpuTerm />。

这和传统卖软件完全不同。传统软件卖出去之后，下一次收入要等升级换代；订阅软件则每年都能续费。

### 3. AI 最容易先嵌进这里

Copilot 为什么重要？不是因为它是一个聊天机器人，而是因为它直接嵌进 Word、Excel、PowerPoint、Outlook、Teams 这些工作场景里。

如果一个 AI 产品能帮企业员工写邮件、总结会议、做 PPT、整理 Excel、查询内部知识库，它就不是“玩具”，而是可以提高办公效率的工具。

所以 Microsoft 做 AI 的一个关键优势是：它不需要从零找用户。用户已经在 Microsoft 365 里了。

Microsoft 365 Commercial cloud、Consumer cloud、LinkedIn、Dynamics 365 的增速不同，背后的驱动也不同。不要只说“Office 很强”，要进一步问：是 seat 增长、<ArpuTerm /> 增长，还是 Copilot / E5 这类高阶套餐在驱动？

## 业务线二：Intelligent Cloud

这个板块是 Microsoft 当前最重要的增长引擎。

它主要包括：

- Azure and other cloud services
- GitHub cloud services
- Nuance Healthcare cloud services
- 虚拟桌面
- SQL Server、Windows Server、Visual Studio 等服务器产品
- 企业支持和行业解决方案

FY2025 Intelligent Cloud 收入 $106.3B，同比增长 21%；经营利润 $44.6B，同比增长 18%。其中 Azure and other cloud services FY2025 增长 34%。

到了 FY2026 Q3，趋势更强：

| 指标 | FY2026 Q3 |
| --- | ---: |
| Microsoft 总收入 | $82.9B，+18% |
| Microsoft Cloud 收入 | $54.5B，+29% |
| Intelligent Cloud 收入 | $34.7B，+30% |
| Azure and other cloud services 收入增速 | +40% |
| AI business 年化收入 run rate | 超过 $37B，+123% |

这几个数字说明一件事：**市场买 Microsoft，不只是买 Office 的稳定现金流，也是在买 Azure 和 AI 基础设施的增长。**

FY2026 Q3 里，AI business 年化收入 run rate 已经超过 $37B，commercial <FinanceTerm term="RPO" /> 增至 $627B，Microsoft Cloud revenue 达到 $54.5B、同比增长 29%。这些数字说明云和 AI 需求仍然很强。

但另一边，Microsoft Cloud <FinanceTerm term="Gross Margin" label="gross margin" /> 降到 66%。这说明 AI 基础设施不是没有成本的增长：客户需求越强，Microsoft 越需要提前投入数据中心、GPU、网络和电力，短期会压低云毛利率。

### Azure 对 Microsoft 的价值

Azure 的价值不只是“云收入增长快”。它对 Microsoft 至少有四层价值：

1. **基础设施层**：企业把计算、存储、数据库、网络、安全迁到 Azure
2. **开发者层**：GitHub、Visual Studio、Azure DevOps 帮 Microsoft 抓住开发者
3. **AI 层**：Azure 提供 GPU、模型服务、AI Agent 平台，承接企业 AI 需求
4. **应用层**：Microsoft 365、Dynamics、Copilot 都可以和 Azure 结合

Amazon AWS 很强，但 Microsoft 的优势在于企业客户关系和应用层入口。很多企业已经买了 Microsoft 365、Windows Server、SQL Server、Active Directory、安全产品，再买 Azure 是自然延伸。

这就是 Microsoft 的“全栈企业云”逻辑。

如果要单独解释 Azure，不能只看 Microsoft 的总收入图，而要看 Intelligent Cloud 板块图。它能告诉你：Azure 拉动收入高增长，但经营利润的扩张速度没有收入那么快，因为 AI 和云基础设施成本也在上来。

### 为什么 Azure 毛利率会承压

FY2025 年报里有一个重要细节：Microsoft Cloud gross margin percentage 下降到 69%，原因是 AI 基础设施扩张带来的影响，部分被 Azure 效率提升抵消。

这说明 AI 不是纯粹的免费增长。AI 需要大量 GPU、数据中心、电力、网络设备和折旧。收入增长很快，但<FinanceTerm term="资本开支" />和成本也会很高。

讲完 Azure 的增长，还必须接着看 CapEx。因为 AI 云不是轻资产增长：数据中心、GPU、网络设备会先进入资本开支，未来能否转化成足够高的云收入和利润，才是投资判断的关键。

所以研究 Microsoft，不能只看 Azure 增速，还要看：

- AI 收入增长能否覆盖资本开支

- GPU 供给是否足够
- 企业客户是否真的把 AI 应用跑起来
- OpenAI 相关投入是否能带来长期回报

## 业务线三：More Personal Computing

这个板块名字听起来不如 Cloud 性感，但不能忽视。

它包括：

- Windows and Devices
- Gaming：Xbox、Game Pass、Activision Blizzard
- Search and news advertising

FY2025 More Personal Computing 收入 $54.6B，同比增长 7%；经营利润 $14.2B，同比增长 18%。

这部分业务的特点是：增长没有 Azure 快，但它有几个战略价值。

More Personal Computing 的收入增长弱于云业务，但经营利润并没有同步恶化。这说明它更像成熟现金流和入口型资产，而不是高增长核心引擎。

### Windows 是入口，不只是操作系统

Windows OEM 收入会受到 PC 出货周期影响，所以它不是高确定性增长业务。但 Windows 仍然是企业和个人 PC 的核心入口。

有了 Windows，Microsoft 才能更自然地推广 Edge、Bing、Copilot、Microsoft 365、OneDrive、安全产品等。

所以 Windows 的投资价值不只是收入，而是生态入口。

### Gaming 是内容和订阅实验场

Microsoft 收购 Activision Blizzard 之后，Gaming 变得更重要。Xbox 硬件本身不是重点，真正重要的是内容、订阅、云游戏和用户时长。

Game Pass 类似“游戏版 Netflix”，如果做成，会把一次性买游戏变成订阅收入。但游戏行业内容成本高、爆款不稳定，所以它的确定性不如 Microsoft 365。

### Search 广告是 AI 分发入口

Bing 很长时间都不是 Google Search 的真正威胁。但 AI 搜索出现后，搜索广告重新变成 Microsoft 的战略入口。

FY2025 Search and news advertising revenue excluding TAC 增长 20%，FY2026 Q3 也增长 12%。规模还远小于 Google，但如果 AI 改变搜索入口，Microsoft 至少有了一个可以继续下注的位置。

Windows、Gaming、Search advertising 虽然放在同一个板块里，但业务属性并不一样：Windows 和 Gaming 更受 PC 周期、内容供给影响，Search advertising 则更像 AI 搜索带来的战略选择权。

## Microsoft 的历史主线：从软件授权到云和 AI

研究 Microsoft，不能只看最新财报。它的历史转型非常重要。

### 第一阶段：Windows + Office 时代

早期 Microsoft 的核心是 Windows 和 Office。PC 越普及，Windows 和 Office 越赚钱。这是典型的软件授权模式：一次销售、高毛利、强生态。

但这个模式有问题：如果 PC 增长放缓，收入增长也会放缓。

### 第二阶段：企业订阅时代

Office 365 把一次性软件授权变成订阅。这是 Microsoft 过去十几年最重要的商业模式升级之一。

订阅让收入更稳定，也让 Microsoft 可以持续追加云存储、安全、Teams、Copilot 等功能。

### 第三阶段：Azure 云时代

Satya Nadella 上任后，Microsoft 的战略核心从 Windows 优先，转向 Cloud first、Mobile first，后来进一步转向 AI 和 cloud platform。

这一步很关键。因为如果 Microsoft 只守着 Windows，它会被移动互联网时代边缘化。但 Azure 让它重新站到企业 IT 基础设施的核心。

### 第四阶段：AI 平台时代

现在的 Microsoft，试图把 AI 嵌进所有业务：

- Azure 提供 AI 基础设施
- GitHub Copilot 改变开发者工作流
- Microsoft 365 Copilot 改变办公工作流
- Dynamics Copilot 改变销售和客服工作流
- Windows Copilot 改变 PC 使用入口

这就是 Microsoft 当前战略的核心：**不是单独卖一个 AI 产品，而是把 AI 变成所有企业软件和云服务的增值层。**

## 怎么判断 Microsoft 的增长质量

研究 Microsoft，重点看几个指标：

1. **Azure and other cloud services 增速**
2. **Microsoft 365 Commercial cloud 增速**
3. **Microsoft Cloud gross margin**
4. **AI business run rate**
5. **CapEx 和折旧压力**
6. **商业剩余履约义务（commercial RPO）**
7. **回购和分红**

FY2026 Q3 里，commercial remaining performance obligation 增加 99% 到 $627B。这个数字可以理解成未来合同收入的蓄水池，说明企业客户已经签下大量未来云和软件服务合同。

这类 backlog 对 Microsoft 很重要，因为它增加了未来收入的可见度。

## Microsoft 的风险

Microsoft 是好公司，但不是没有风险。

### 1. AI 资本开支过高

AI 基础设施需要大量投入。如果企业客户 AI 应用落地速度不如预期，资本开支可能先压低自由现金流，收入兑现却滞后。

### 2. Azure 竞争加剧

Azure 面对 AWS、Google Cloud，以及部分企业自建或混合云方案。云计算是好行业，但不是没有竞争。

### 3. OpenAI 关系和投资回报

Microsoft 和 OpenAI 的合作是优势，也是变量。需要持续观察 OpenAI 相关收入、成本、投资损益和战略关系变化。

### 4. Copilot 商业化不及预期

Copilot 如果能提高 Microsoft 365 <ArpuTerm />，就是巨大利好。但如果用户觉得价格太高、使用频率不够，商业化会低于预期。

### 5. 监管风险

Microsoft 体量太大，云、AI、游戏、企业软件都可能受到监管关注。

## 最后怎么给 Microsoft 下结论

Microsoft 的核心投资逻辑可以总结成一句话：

> 用 Microsoft 365 和企业客户关系提供稳定现金流，用 Azure 和 AI 提供增长，用 Windows、GitHub、LinkedIn、Dynamics、Gaming 扩大生态。

它不是最高速增长的科技公司，但它非常罕见地同时具备：

- 高利润率
- 强现金流
- 企业客户锁定
- 云计算增长
- AI 分发入口
- 多业务协同

研究 Microsoft 时，最重要的问题不是“它是不是好公司”，而是：

1. 当前估值是否已经充分反映 Azure 和 AI 增长？
2. AI 投资能否转化成高质量自由现金流？
3. Copilot 是否真的能提高 Microsoft 365 的 <ArpuTerm />？
4. Azure 能否继续保持高于行业平均的增长？

如果这些问题的答案是肯定的，Microsoft 就是典型的高质量复利公司。如果答案开始变弱，即便它仍然是好公司，估值也可能需要下修。
